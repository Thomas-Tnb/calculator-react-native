Requisitos :

- App deve ser mobile
- App deve ter uma tela inicial que contenha um teclado numérico para o usuário digitar os números da operação e exibir também opções de operadores de : soma, subtração, multiplicação e divisão.
- App deve armazenar 2 números inteiros
- App deve realizar operação matemática com operador escolhido
- App deve permitir realização de várias operações ( não simultâneas ) reutilizando resultados
